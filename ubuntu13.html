UBUNTU 13 SERVER

Download .img

This is the Image File for Ubuntu 13 Server PowerPC and it has been modified to boot on the Nintendo Wii.

That file will two partitions: a fat32 partition and a ext4 partition. The fat32 partition will contain the Linux loader for the Homebrew Channel.

Just flash this image to an SD Card (2GB or more needed!)

Username - ubuntu

Password - ubuntu

Swapfile has already been setup but is small and can be made bigger if you wish. Swapfile is needed to even run basic tasks but was kept small in order to keep download size as small as possible.

Its as simple as using the "whiite-ez-wifi-config" file located in "/home/ubuntu" and setting up for wifi!

If that doesnt work please use the command "sudo nano /etc/network/interfaces" and add the ssid and passkey yourself.
