HOW TO BOOT

You will see 3 Kernel Choices, by default it boots the stable v4.20 kernel from USB, Older kernels can be compiled and work but are abandonded for this release.

Under the USB option you see the SD card option which is the option used for the AIO Image.

USB Ethernet and WiFi NOW WORK!.

Framebuffer support depends on the release you are using. Newest kernel supports Wii GPU Framebuffer.

The USB Image currently has the AIO swap in fstap. That can be fixed by changing it to /dev/mmcblk0p3 to mmcblk0p2 if using the USB Only BOOTFS.
